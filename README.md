## 📖 Grammatik-Referenz

### Konditional (höfliche Bitten)
```
io    → vorrei     (ich hätte gerne)
tu    → vorresti
lui   → vorrebbe
noi   → vorremmo
voi   → vorreste
loro  → vorrebbero
```

### Imperativ formell (Richtungen geben)
```
andare  → vada    (gehen Sie)
girare  → giri    (biegen Sie ab)
prendere → prenda (nehmen Sie)
continuare → continui (fahren/gehen Sie weiter)
attraversare → attraversi (überqueren Sie)
```

### Direkte Objektpronomen
```
lo  = ihn / es (mask. Singular)
la  = sie / es (fem. Singular)
li  = sie (mask. Plural)
le  = sie (fem. Plural)
```
